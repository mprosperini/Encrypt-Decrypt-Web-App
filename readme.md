## How it works?

The following characters will be changed after presing **"Encrypt"** button and viceversa after pressing **"Decrypt"** button. 
|  Original | Encrypted  |
| ------------ | ------------ |
| a  | ai  |
|  e | enter  |
|  i |  imer |
|  o | ober  |
|  u |  ufat |
